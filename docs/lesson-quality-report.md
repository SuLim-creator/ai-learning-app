## 레슨별 분석

### 01. 벡터란 무엇인가?

- **난이도 분포**: easy 1 / medium 0 / hard 0
- **사실 오류**: 없음. 256×256×3=196,608, 내적 11, 크기 √(3²+4²)=5 모두 정확.
- **개선점**:
  - 퀴즈가 1개뿐 — 다른 레슨(3개)과 일관성 부족. 최소 2~3개 추가 권장 (예: "내적 [2,3]·[1,4]=?", "단위벡터의 크기는?")
  - "Word2Vec, BERT가 수백 차원" → BERT는 768~4096차원이므로 "수백~수천 차원"으로 수정
  - "스칼라 곱" 용어 도입 전 간단한 풀어쓰기 권장

### 02. 행렬과 행렬 연산

- **난이도 분포**: easy 1 / medium 0 / hard 0
- **사실 오류**: 없음. (3×4)×(4×2)=3×2 정확.
- **개선점**:
  - **구조적 중복**: 레슨 04(행렬 정의)·05(행렬 곱셈)와 내용 거의 완전 중복. 02를 폐기하거나 "선형대수 개요" 위치로 재배치 권장
  - 퀴즈 1개 → 3개로 보강 필요

### 03. 미분과 경사하강법

- **난이도 분포**: easy 0 / medium 1 / hard 0
- **사실 오류**: 없음. 학습률이 크면 발산한다는 설명 정확.
- **개선점**:
  - **주제 중복**: 레슨 06(경사하강법)이 동일 주제를 더 풍부하게 다룸 → 03을 "순수 미분 개념"만 다루도록 범위 축소 권장
  - 미분 정의를 "Δy/Δx 극한"으로 한 줄 수식 직관 추가
  - 퀴즈 1개 → 3개로 보강 (예: "f(x)=x²의 x=2에서 미분값은?")

### 04. 행렬이란 무엇인가

- **난이도 분포**: easy 3 / medium 0 / hard 0
- **사실 오류**: 없음. 50명×3특성=50×3, 4×7=28개 정확.
- **개선점**:
  - A[1][2]=90 표기 시 1-인덱스(수학) vs 0-인덱스(프로그래밍) 혼동 주의. "수학 표기(1부터 시작)를 사용합니다" 명시 권장
  - 퀴즈 1번 오답 C "순서가 있는 숫자들의 목록"이 벡터 정의와 유사 → 해설에 "이것은 벡터의 정의"라고 명시 권장

### 05. 행렬 곱셈

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음. 2×2 예시 계산(19, 22, 43, 50) 및 단위행렬 설명 모두 정확.
- **개선점**:
  - 퀴즈 1(결과 크기)이 레슨 02 퀴즈와 내용 중복 → 02 통합/폐기 시 자연 해결
  - "단위행렬" 용어가 본문에서 정의되기 전에 퀴즈에 등장 → 퀴즈 직전 단계에서 1줄 정의 추가

### 06. 경사하강법

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음. w_new=4-0.1×8=3.2, 3.2→2.56 계산 정확.
- **개선점**:
  - "학습률 0.001~0.01" → 실제로는 1e-5~1e-1까지 다양. "Adam 기본값 0.001 등 흔히 사용하는 범위"로 완화 권장
  - 지역 최솟값(local minimum) 함정을 1줄 언급 권장

### 07. 활성화 함수

- **난이도 분포**: easy 2 / medium 1 / hard 0
- **사실 오류**: 없음. ReLU(-3)=0, Sigmoid 범위 (0,1), Softmax 합=1 정확.
- **개선점**:
  - Sigmoid가 현재 은닉층에서 거의 사용되지 않는 이유(기울기 소실) 1줄 추가
  - "유리 렌즈 굴절" 비유는 비선형성 전달이 불완전 → "고무판/지점토 변형" 비유로 교체 검토

### 08. 손실 함수란 무엇인가

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음. MSE 계산 (1+4+1)/3=2.0, Cross-Entropy 설명 정확.
- **개선점**:
  - **표기 불일치**: 본문 표에서 "실제값-예측값" 부호로 표기했으나 퀴즈 1에서는 "예측값-실제값"으로 다름 → **부호 정의 통일 필요**
  - Cross-Entropy 직관 수치 예 추가 권장: -log(0.1)≈2.3 vs -log(0.9)≈0.1
  - 퀴즈 3번 오답 D "손실 함수 없이도 학습 가능"은 너무 명백한 오답 → "L1 손실로 대체 가능" 같은 그럴듯한 미끼로 교체

### 09. 함수란 무엇인가

- **난이도 분포**: easy 2 / medium 1 / hard 0
- **사실 오류**: 없음. f(2)=11, f(g(3))=19, f(g(3))=16 모두 정확.
- **개선점**:
  - 본문 예시(f(g(3))=19)와 퀴즈(f(g(3))=16)가 다른 함수를 씀. 학생 혼란 방지를 위해 "퀴즈는 본문과 다른 함수입니다" 안내 1줄 추가 권장

### 10. 확률과 불확실성

- **난이도 분포**: easy 2 / medium 1 / hard 0
- **사실 오류**: 없음. 3/10=0.3, 확률 합=1 정확.
- **개선점**:
  - "GPT가 '고양이'라고 답할 때 92%..." → GPT는 언어 모델이므로 이미지 분류 예시에 어울리지 않음. **"이미지 분류 AI" 또는 "Vision 모델"로 표현 수정**
  - "콜모고로프 공리" 용어 → 비전공자 배려로 "확률의 기본 약속"으로 풀어쓰기
  - 퀴즈 2번 오답 D "토끼를 제거하고 다시 계산" 어색함 → 더 그럴듯한 미끼로 교체

### 11. 분포: 데이터의 형태

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음. μ+2σ 단측 2.5%, μ+3σ 단측 0.15% 모두 정확.
- **개선점**:
  - "AI 이상 탐지는 3σ 이상 벗어나면 이상으로 분류" → 단순화 표현. "전통적 통계 방법 중 하나는"이라는 한정 추가 권장 (실제로는 Isolation Forest, autoencoder 등 다양)
  - 정규분포 외 베르누이, 푸아송 분포를 1줄 언급해 학습 폭 확장
  - 퀴즈 3번 오답 D "분포를 따르지 않으면 이상치가 없다" 너무 명백 → "비정규 데이터는 IQR 등 다른 방법이 필요하다" 등으로 교체

### 12. 조건부 확률과 베이즈

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음. 베이즈 의료 예시 P(병|양성)=50% 정확. 스팸 필터 ≈0.87 정확.
- **개선점**:
  - 베이즈 정리 분모 계산에 전확률 법칙 1줄 추가 권장: `P(B) = P(B|A)P(A) + P(B|¬A)P(¬A)`
  - 가능도(likelihood)/사후 확률(posterior) 용어 일관성 양호. 유지.

### 13. 1단계 총정리: AI를 위한 수학 기초

- **난이도 분포**: easy 1 / medium 2 / hard 0
- **사실 오류**: 없음.
- **개선점**:
  - **prerequisites 불완전**: 현재 09, 10, 12만 포함 → 1단계 총정리이므로 01~12 전체(또는 핵심 6개 이상)로 확장 필요
  - 개념 지도(s1)에서 **활성화 함수, 손실 함수 누락** → 추가 권장
  - 종합 정리임에도 **hard 퀴즈 0개** → 종합 추론 퀴즈 1개 추가 권장 (예: "MSE와 Cross-Entropy 중 회귀 문제에 적합한 것은? 이유는?")
